AVADoma

Domain name registration for IPFS decentralized applications on the Avalanche blockchain.

How it works?

1. Search domain name

The user search domain name they like


2. Set CID IPFS

The user uploads his decentralized application to IPFS and specifies its CID


3. Registration period

The user selects the domain name registration period


4. Send transaction

The user pays for transaction and domain is registered in the Avalanche blockchain


How do I transfer a domain name to another user?

1. Update Transfer field

In the Transfer field, specify the address of the new domain owner


2. New domain price

The user will be able to get a domain name when they pay the specified amount to AVAX


3. Updating domain

The user updates data in the domain smart contract


4. Payment and transfer

The user to whom the domain name was transferred can now receive it by paying the amount specified by the seller


Attracting users

A random domain name will be played every day


Browser extension

Interaction with IPFS occurs through a browser extension
